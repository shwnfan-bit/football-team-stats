## 📖 关键概念速查

### Vercel 是什么？
Vercel 是 Next.js 官方推荐的部署平台，提供全球 CDN、自动 HTTPS、零配置部署等功能。

### 部署需要多久？
首次部署：2-5 分钟
后续部署：1-2 分钟

### Vercel 免费吗？
是的！Vercel 提供慷慨的免费额度：
- 100GB 带宽/月
- 无限项目数
- 全球 CDN
- 自动 HTTPS

### 需要域名吗？
不需要！Vercel 会自动生成免费域名：
`https://your-project-name.vercel.app`

当然，你也可以配置自定义域名。

## 🔧 环境变量配置

### DATABASE_URL 格式
```
postgresql://username:password@host:port/database
```

### 配置位置
项目配置页面 → Environment Variables → New Variable

### 必须配置吗？
是的！如果不配置 DATABASE_URL，应用无法连接数据库。

## 🌍 推荐部署区域

**中国用户推荐**：Hong Kong (hkg1)

**设置方法**：
- 网站：项目设置 → General → Regions → Hong Kong (hkg1)
- CLI: `vercel regions set hkg1`

## 💡 使用提示

### 建议阅读顺序
1. 先看快速开始，了解整体流程
2. 再看可视化图示，理解界面布局
3. 最后看详细步骤，掌握具体操作

### 常用操作位置
- 环境变量：Settings → Environment Variables
- 部署区域：Settings → General → Regions
- 构建日志：Deployments → 点击部署 → Build Logs
- 自定义域名：Settings → Domains

### 记住这些快捷键
- `vercel login` - 登录 Vercel
- `vercel` - 部署到预览环境
- `vercel --prod` - 部署到生产环境
- `vercel logs` - 查看日志
