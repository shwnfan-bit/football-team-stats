## 🎯 关键操作位置

### 环境变量配置

```
项目设置 → Settings → Environment Variables → Add New
```

### 修改部署区域

```
项目设置 → Settings → General → Regions → 选择 Hong Kong (hkg1)
```

### 查看构建日志

```
Deployments → 点击具体部署 → Build Logs
```

### 重新部署

```
Deployments → 点击旧部署 → Redeploy
```

### 配置自定义域名

```
Settings → Domains → Add Domain
```
